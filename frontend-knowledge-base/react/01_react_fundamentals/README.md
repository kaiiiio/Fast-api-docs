## Key Concepts

- **JSX**: JavaScript XML syntax
- **Components**: Building blocks of React
- **Props**: Data passed from parent
- **State**: Component-specific data
- **Lifecycle**: Component mounting, updating, unmounting
- **Events**: User interactions
