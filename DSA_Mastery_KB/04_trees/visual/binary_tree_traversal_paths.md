# Binary Tree Traversals

Tree:
```
      1
     / \
    2   3
```

## 1. Preorder (Root -> Left -> Right)
`1 -> 2 -> 3`
Use: Copying tree.

## 2. Inorder (Left -> Root -> Right)
`2 -> 1 -> 3`
Use: Sorted order for BST.

## 3. Postorder (Left -> Right -> Root)
`2 -> 3 -> 1`
Use: Deleting tree (children first).
